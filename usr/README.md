# Kruthika Professional Resume App

A professional Flutter application showcasing the resume and portfolio of Kruthika, a B.Com graduate and entry-level finance/accounting professional.

## Overview
This app is a digital resume containing:
- **Profile Summary**: Highlighting her background as a B.Com graduate and her training with the Quest Alliance program by Accenture.
- **Contact Details**: Easy access to email and phone.
- **Skills**: Showcasing expertise in Tally, MS Word, Microsoft Excel, and Communication.
- **Education**: Detailed academic background.
- **Cover Letter Template**: A customized job application letter specifically tailored for entry-level finance, accounting, or back-office roles.

## Tech Stack
- **Flutter**: Cross-platform framework for building native-compiled applications for mobile, web, and desktop from a single codebase.
- **Dart**: Programming language used by Flutter.

## How to Run
1. Ensure you have the Flutter SDK installed.
2. Clone this repository and navigate to the project root.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the app on your preferred connected device or emulator.
